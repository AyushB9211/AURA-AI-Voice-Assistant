# 🤖 AURA – AI Voice Assistant (Python Project)

Build your own Jarvis-like AI assistant using Python.

This project demonstrates Speech Recognition, Text-to-Speech, API integration and Desktop Automation.

---

## ✨ Features

🎤 Voice + Text hybrid commands  
🔎 Google Search automation  
▶ YouTube video playback  
📚 Wikipedia search  
🌦 Real-time Weather updates  
📰 Latest News headlines  
⏰ Time & Date commands  
😂 Joke generator  
💬 Friendly AI voice responses  

---

## 🛠 Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- OpenWeather API
- News API
- Wikipedia API

---

## 🚀 How to Run

1️⃣ Install requirements

pip install -r requirements.txt

2️⃣ Create .env file from .env.example

3️⃣ Run the assistant

python main.py
